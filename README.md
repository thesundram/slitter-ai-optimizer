# Slitter AI Optimizer

A modern web application for optimizing coil slitting operations in steel industries with intelligent pattern generation and line balancing.

## Features

- **RM Coil Management** - Track and manage raw material coils
- **Sales Order Processing** - Handle customer orders efficiently
- **Line Specification** - Configure production line parameters
- **AI-Powered Optimization** - Intelligent slitting pattern generation
- **Forecasting** - Predictive analytics for production planning
- **Real-time Visualizations** - Interactive charts and dashboards

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Charts**: Recharts
- **State Management**: React Context
- **Form Handling**: React Hook Form + Zod

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd slitter-ai-optimizer
```

2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `pnpm dev` - Start development server
- `pnpm build` - Build for production
- `pnpm start` - Start production server
- `pnpm lint` - Run ESLint
